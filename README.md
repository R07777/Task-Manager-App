# Task Manager App

## Features
- User Authentication (Login/Register)
- JWT Authentication
- CRUD Operations (Tasks)

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Run Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
